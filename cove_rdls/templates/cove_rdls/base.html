    {% blocktrans %}About the Risk Data Library Standard{%endblocktrans%}

    {% blocktrans %}The Risk Data Library Standard (RDLS) is an open data standard to make it easier to work with disaster and climate risk data. It provides a common description of the data used and produced in risk assessments, including hazard, exposure, vulnerability, and modelled loss, or impact, data.{%endblocktrans%}

    {% blocktrans %}The RDLS provides a unique way to create, store, exchange and use disaster different risk information together. It is at the core of the Risk Data Library, a suite of open-source tools to work with disaster and climate risk data.{%endblocktrans%}

    {% blocktrans %}Learn more at the RDLS documentation website.{%endblocktrans%}

    Large datasets

    If your data file is larger than 10MB, this analysis may take some time. You can process a subset of your submitted data using Sample mode (see below). Alternatively you can install our data review library to analyse files via your command line interface.

    Sample mode validates and analyses the data from a subset of the submitted RDLS datasets. Datasets of all four types (hazard, exposure, vulnerability and loss) are represented in the sample. Sampling is not random: if an unchanging data is run through the data review tool multiple times, the same datasets will form the sample each time. Conformance tests that require processing of the complete data are not run in Sample mode.
